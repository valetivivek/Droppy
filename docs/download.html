Get Droppy

Droppy is free and open-source. Built entirely in Swift. Requires macOS 14 Sonoma or later.

Recommended

Homebrew

Install via Homebrew for automatic updates and easy management.

Install Droppy

brew install --cask iordv/tap/droppy

Don't have Homebrew? Install it first

/bin/bash -c "$(curl -fsSL https://brew.sh/install.sh)"

Direct Download

Download the DMG installer directly from GitHub releases.

Step 1: Download

Download Droppy.dmg

Step 2: Run in Terminal

xattr -d com.apple.quarantine ~/Downloads/Droppy.dmg

Step 3: Open the DMG and install
