## What's Droppy?

Ever wish your Mac's notch did something useful? Droppy turns it into a file shelf, clipboard manager, media controller, and more â€” all in one beautiful interface.

**No notch? No problem.** Droppy creates a sleek Dynamic Island-style pill that works on any Mac.

### The basics:

- **Drag files to the notch** â€” they slide into a hidden shelf, ready whenever you need them
- **Jiggle your mouse while dragging** â€” a floating basket appears anywhere on screen  
- **Full clipboard history** â€” search, pin favorites, extract text from images, drag items out
- **Media controls in the notch** â€” album art, seek slider, shuffle/repeat, Spotify integration

## Install

### Homebrew (recommended)
```bash
brew install --cask iordv/tap/droppy
```

### Manual Download
1. Download [**Droppy.dmg**](https://github.com/iordv/Droppy/releases/latest)
2. Clear quarantine (required for unsigned apps):
   ```bash
   xattr -rd com.apple.quarantine ~/Downloads/Droppy-*.dmg
   ```
3. Open the DMG and drag Droppy to Applications

> ðŸ’¡ Homebrew handles quarantine automatically and makes updates easy.
